AlphaRank

A systematic NSE weekly-rotation equity strategy — multi-horizon LightGBM, fully risk-managed, fully automated.

LightGBM Cross-sectional ranking ~360 NSE stocks Walk-forward tested Python · Flask

What it does

Every week the model scores ~360 NSE stocks, ranks each against all the others, and holds the top 7. It blends 5-day, 10-day and 20-day return forecasts, sizes positions by risk-parity, and protects capital with VIX-adaptive stops, sector caps and a transaction-cost gate. It runs end-to-end on its own: daily data refresh, weekly rebalance, live dashboard. Currently in paper / dry-run mode.

System architecture

Data sources → daily pipeline → LightGBM model → risk gates → execution → dashboard. Daily data refresh, weekly rebalance, fully automated.

Under the hood

  • Multi-horizon model — blends 5/10/20-day return forecasts
  • ~30 features — momentum, volatility, RSI/MACD, volume, macro, sentiment, peer-ranks
  • Cross-sectional ranking — picks relative winners, market-neutral by design
  • Risk-parity sizing with 35% sector / 18% single-stock caps
  • VIX-adaptive stops — tight in panic, loose when calm
  • Transaction-cost gate — skips trades that can't beat their cost
  • Incremental data cache — daily refresh in seconds, not minutes
  • Walk-forward backtest with purge gaps + Nifty benchmark
